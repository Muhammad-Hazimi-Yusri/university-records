# ELEC3224 - Guidance, Navigation and Control 2023-24

## Module overview

This module will be first offered in the 2019/20 academic year. This module will provide a basic grounding in navigation guidance and control with particular aspects on the processing of the signals involved and overall system integration.

## Aims

This module aims at providing a basic understanding of navigation guidance and control, with special attention to the signal processing aspect and overall system integration.

## Learning Outcomes

- **A1:** Fundamentals of the various guidance techniques and their properties.
- **A2:** Position and attitude estimation.
- **B1:** System design and validation.
- **D1:** Examples of current and planned implementations and applications.
- **E1:** Overall system integration.

## Syllabus summary

- System description and model building. - Model validation. - Sensing and actuating systems and their relative strengths and limitations. - Fundamentals of classical guidance algorithms, e.g., proportional navigation and command to line of sight. - Estimation of position and attitude. - Introductions to autonomy. - Guidance and navigation for autonomous systems.
